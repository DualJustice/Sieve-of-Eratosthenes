# Sieve of Eratosthenes
 Made by: DualJustice (3/18/2021)
 
 
 This program is made to be run on an Arduino.
 Note that if "No Line Ending" is NOT selected in your serial monitor, you will incounter a false loop when an integer that is < 2 is entered.
 As such, I recommend that the "No Line Ending" option is selected.
